# Screenshots for Chrome Web Store

Required dimensions: 1280×800 or 640×400 (recommended: 1280×800)

## Needed Screenshots

1. **upload.png** — Upload screen with drag-and-drop zone
2. **styles.png** — Style preset selection
3. **generating.png** — Generation in progress with progress indicator
4. **results.png** — Grid of generated headshots
5. **share.png** — Share modal with social buttons
6. **settings.png** — Settings panel with toggles

## Marquee Image
- **marquee.png** — 1400×560 promotional banner

## Tips
- Use Chrome DevTools → Device Toolbar → Dimensions: 1280×800
- Capture with clean browser chrome
- Ensure text is readable at thumbnail size
- Show diversity in example photos
